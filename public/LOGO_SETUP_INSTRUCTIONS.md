# Sentient Portal Logo Setup Instructions

## Required Action

Please add your Sentient Portal logo image to this directory with the following filename:

**sentient-portal-logo.png**

## Logo Specifications

The logo image you provided shows:
- Fingerprint at the center representing human identity
- DNA helix strands forming an "S" shape
- Teal/silver metallic gradient
- Text: "Sentient Portal"
- Tagline: "verified human-only web | sentport.com"

## File Requirements

- **Format**: PNG with transparency recommended
- **Dimensions**: Minimum 800x800px for best quality across all placements
- **File size**: Optimized for web (under 500KB recommended)
- **Background**: Transparent background preferred for versatility

## Where the Logo Appears

Once you add the logo file, it will automatically appear in:

1. **Browser Tab (Favicon)** - Shows in the browser tab
2. **Header** - Top left corner of every page (40px height)
3. **Homepage Hero** - Large centered logo (192px height)
4. **Footer** - Bottom of every page (64px height)
5. **Sign In Page** - Above the welcome message (96px height)
6. **Sign Up Page** - Above the form (96px height)
7. **Reset Password Page** - Above the form (96px height)
8. **About Page** - Top of the page (160px height)
9. **Manifesto Page** - Header section (128px height)

## Installation

Simply save your logo image as:
```
/public/sentient-portal-logo.png
```

The application is already configured to use this file.

## Apple Touch Icon

For best iOS/mobile experience, also consider adding:
- `/public/apple-touch-icon.png` (180x180px)

This ensures the logo appears correctly when users add your site to their home screen.

## Testing

After adding the logo:
1. Refresh your browser
2. Check the browser tab for the favicon
3. Navigate through the site to see the logo in all placements
4. Test on mobile devices to ensure proper scaling
